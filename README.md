# Bank-Loan-Report-PowerBI
Power BI dashboard analyzing bank loan data including approvals, funded amount, borrower profiles and loan performance.

## Project Overview
This project presents an interactive Power BI dashboard analyzing bank loan data to identify approval trends, funded amounts, borrower profiles, and loan performance.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- CSV Dataset

## Key Insights
- Total Loan Applications
- Total Funded Amount
- Loan Approval vs Rejection Rate
- Borrower Income Distribution
- Loan Status Analysis

## How to Use
1. Download the .pbix file
2. Open it using Power BI Desktop
3. Explore the interactive dashboard
